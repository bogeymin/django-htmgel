{% comment %} breadcrumbs.html ================ Display breadcrumb navigation. **breadcrumbs** (iterable) A list of breadcrumb objects. The object must provide a ``title`` attribute and implement ``get_absolute_url()``. **include_home** (bool) Indicates whether a home link should be added automatically. **label** (str) Optional label for the navigation. May not be used by all frameworks. Example: .. code:: {% include "htmgel/breadcrumbs.html" with breadcrumbs=objects include_home=True %} {% endcomment %} {% if HTML_FRAMEWORK == "bootstrap3" %} {% include "htmgel/bootstrap3/breadcrumbs.html" %} {% elif HTML_FRAMEWORK == "foundation6" %} {% include "htmgel/foundation6/breadcrumbs.html" %} {% else %}
    {% if include_home %}
  1. {% trans "Home" %}
    2. {% endif %} {% for crumb in breadcrumbs %} {% if forloop.last %}
  2. {{ crumb.title }}
    3. {% else %}
  3. {{ crumb.title }}
    4. {% endfor %}
{% endif %}
