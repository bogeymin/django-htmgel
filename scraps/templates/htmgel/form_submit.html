{% comment %} form_submit.html ================ Displays a submit button. This is useful when you are manually controlling the ``
`` tag. A generic means for display most types of forms. **button_text** (str) The text displayed for the button. Defaults to ``Submit``. **cancel_text** (str) The label for the cancel link. Defaults to ``Cancel``. **cancel_url** (str) The URL for the cancel link. If omitted, no link is provided.
